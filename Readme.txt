Khayalan GIFShot is program to make a capture from webcam into GIF animation format.
GIFShot is easy, powerfull, and user friendly interface, making GIF animation made easy!


Features:
____________________

* Capture directly to GIF animation without another extrenal program.
* Setting up frame speed of your animation.
* Compression with LZW, make your animation file smaller.
* Limit the number of frames.
* Resize animation directly.
* Reversal frame addition, make your animation lives!
* Result can send it by e-mail or copy to portable device.
* Able to pause capture, make more experience capture.
* And any others...
