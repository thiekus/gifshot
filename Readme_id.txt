Program ini mampu membuat rekaman dengan webcam dengan hasil akhir berupa file animasi GIF secara langsung tanpa program bantuan lainnya.


Fitur-fitur program:
____________________

* Merekam langsung dengan hasil animasi GIF tanpa program bantu lainnya.
* Mengatur kecepatan frame animasi.
* Kompresi dengan LZW, sehingga hasil akhir menjadi lebih kecil.
* Membatasi frame yang akan direkam.
* Mengatur besar hasil secara langsung.
* Menambah frame diakhir secara terbalik, sehingga hasil akhir menjadi tidak kaku.
* Hasil perekaman bisa langsung dikirim via e-Mail ataupun ke flashdisk.
* Bisa pause untuk perekaman yang lebih seru!
* Dan masih banyak lagi...
